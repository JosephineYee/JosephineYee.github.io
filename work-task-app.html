Mobile UI Design - Task App: Todo Checklist & Tracker

Interface design for Todo Checklist & Tracker

Rationale

Here is the reasoning behind my design choices for the interface:

  • A dashboard is accessible from the home screen, allowing users to monitor both completed and pending tasks.
  • Each task includes a selector that enables users to mark tasks as completed.
  • If the user wishes to remove the task, they can do so by clicking on the trashcan icon.
  • I integrated various user-specific features into the menu, including a calendar, notes, and profile options.
  • I selected the color blue because it symbolizes responsibility and induces a calming and relaxing effect on the mind, instilling a sense of confidence and security in the user.
